What is ShopLifter?

ShopLifter is a Content Management System web application that can be launched from any major browser and offers a solution to small business owners looking to start their own eCommerce web site, but not not looking to spend money on more expensive, monthly-paid solutions.

  • Graphical User Interfaces
  • Shopping cart and checkout options with payment acceptance assistance
  • Easy Sales and Promotions management
  • Sales and business report generation
ShopLifter is:
  • A RESTful web service with HTTP and URI contracts
  • Database-driven (SQL) inventory management
  • JavaScript and Java (with Wicket) front end and Java back end architecture
